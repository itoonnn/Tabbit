# Roadmap

## Phase 0: Validation

### Goal

พิสูจน์ว่ากลุ่มเล็ก ๆ จะ check in ต่อเนื่อง และ encouragement จากเพื่อนช่วยให้คนกลับมาทำ habit วันถัดไป

### Key Question To Answer

ผู้ใช้กลับมา check in เพราะ accountability ของกลุ่มจริงหรือไม่

### Scope

- Interview target users
- Test a manual challenge in LINE group chats
- Validate whether people check in consistently
- Identify why people drop off
- Learn what makes a group active or inactive
- Test whether encouragement from friends improves return rate

### Success Signal

- สมาชิกส่วนใหญ่ check in ได้ต่อเนื่องอย่างน้อย 7 วัน
- กลุ่มมี encouragement เกิดขึ้นเองบางส่วน
- ผู้ใช้บอกเหตุผลชัดเจนว่ากลุ่มช่วยให้กลับมาทำต่อ

### Things Not To Build Yet

- Full app
- Complex points system
- Leaderboard
- Multiple habit types
- Personal habit tracker
- LINE LIFF integration

## Phase 1: MVP

### Goal

สร้าง product ที่รองรับ core loop แบบเล็กที่สุด: create group, set shared habit, daily check-in, see progress, encourage, return tomorrow

### Key Question To Answer

MVP ทำให้กลุ่มเริ่ม challenge และ check in ต่อเนื่องได้โดยไม่ต้องพึ่ง founder มากแค่ไหน

### Scope

- Sign in
- Create or join group
- Create one shared habit
- Daily check-in
- Streak
- Group progress view
- Basic encouragement or reaction
- Simple invite flow

Points, rewards และ leaderboard เป็น optional experiments ไม่ใช่ guaranteed MVP features

### Success Signal

- กลุ่มสร้าง challenge ได้เอง
- สมาชิกเข้าใจ check-in flow โดยไม่ต้องอธิบายเยอะ
- Day 7 retention มีสัญญาณดี
- มีการ encourage กันในกลุ่ม

### Things Not To Build Yet

- Advanced gamification
- Marketplace or public communities
- Complex admin tools
- Many notification settings
- Broad task-management features

## Phase 2: Growth Experiments

### Goal

ทดลอง loop ที่ช่วยให้กลุ่มใหม่เกิดง่ายขึ้น กลุ่มเดิมกลับมาเล่น challenge ต่อ และ use case ขยายอย่างมีหลักฐาน

### Key Question To Answer

อะไรทำให้ Tabbit เติบโตจาก challenge แรกไปสู่ repeat usage และ invite loop

### Scope

- Invite loop
- Group challenges
- Points and milestone rewards
- Personal habits
- Group templates
- Notification experiments
- Leaderboard experiment only if validated
- Community use cases
- LINE LIFF integration exploration

### Success Signal

- ผู้ใช้สร้าง challenge ที่สองหลังจบ challenge แรก
- สมาชิกเชิญเพื่อนเข้ากลุ่มเพิ่ม
- Templates หรือ rewards เพิ่ม completion rate อย่างชัดเจน
- Notification ช่วย return rate โดยไม่สร้างความรำคาญ

### Things Not To Build Yet

- Enterprise features
- Large public social network
- Complex monetization
- AI coaching
- Broad project management tools
